This plugin helps to add carousel items in your wordpress website. It is very easy to use with a simple shortcode.

== Description ==

Easy Owl Carousel allows you to add carousel items in your wordpress website. This plugin is based upon Owl Carousel 2. It is very easy to use with a simple shortcode.Just use this code [owl_carousel] to publish your owl carousel.

= Demo =

Go to [Plugin Demo Site](http://creativewebdesign.byethost10.com/plugindemo/owl-carousel/)

= Usage =
Follow the instructions below.

1. Go Owl Carousel menu > Add New.  

1. Give a title and upload image from Set Featured Image Metabox.

1. Create a category and select the category. And publish the post.

1. Add new owl carousel item as much as you want to publish.

1. Use this shortcode in your post or page or widget.

<code>[owl_carousel]</code>

1. To use this shortcode in your template, write 

<code><?php echo do_shortcode('[owl_carousel]'); ?></code>

= More Usage Info =

Use this code to get to publish your carousel from a particulat category.

<code>[owl_carousel category="Your_Category_Name"]</code>



== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

Owl Carousel Menu will appear in admin menu. Add carousel items and create a category. To publish the carousel use shortcode in a post or a page or a template.
